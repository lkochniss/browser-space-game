Ein Nebel ist ein [[POI]] in einem [[Sonnensystem]]. In ihm können sich [[Flotte]] verstecken, wodurch sie von anderen Flotten oder [[Sonde]]n erst viel später erkannt werden. Sie dienen damit der taktischen Kriegsführung und können für einen Hinterhalt genutzt werden, oder um sich vor einer größeren Flotte zu verstecken. Eine Sonde oder eine Flotte muss einen Nebel anfliegen, um die dort versteckten Flotten zu entdecken
